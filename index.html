Sistema simples para gerenciar suas tarefas

Cadastre, edite, exclua e visualize suas tarefas. Projeto para fins de apresentação acadêmica.

Funcionalidades

  • Cadastrar: título, descrição e prazo.
  • Listar: visualização das tarefas.
  • Editar: alterar título, descrição e prazo.
  • Excluir: remover tarefa com confirmação.
  • Salvamento automático: persistência local (simulada).

Simulação local — os dados são salvos no armazenamento local do navegador.

Instruções

  1. Preencha o título e clique em salvar.
  2. Edite ou exclua usando os botões ao lado de cada tarefa.
  3. Os dados são mantidos no navegador (localStorage).
