Funcionalidades

  • Cadastrar tarefas (título, descrição, prazo)
  • Editar e excluir
  • Marcar como concluída
  • Filtros: Todas / Pendentes / Concluídas
  • Contador e gráfico (donut)
  • Persistência local (localStorage)

Como usar

  1. Adicione uma tarefa com título (pressione Enter para enviar).
  2. Marque a checkbox para marcar como concluída.
  3. Use o filtro para mostrar apenas pendentes ou concluídas.
  4. Os dados são salvos automaticamente.
